# 📊 PhonePe Transaction Insights Dashboard

## Project Overview

This project presents an interactive Power BI dashboard inspired by PhonePe transaction analytics. The dashboard provides insights into transaction volume, transaction value, user engagement, and payment success rates through dynamic visualizations.

## Dashboard Preview

![Dashboard](e:\SUMMER INTERNSHIP LPU\PhonePe-Transaction-Insights-Dashboard\Images\Dashboard.png)

## Key Performance Indicators

* Total Transactions
* Total Transaction Value
* Unique Users
* Success Rate
* Month-over-Month Growth

## Features

* Interactive Slicers
* Dynamic KPI Cards
* Transaction Trend Analysis
* Service-wise Transaction Analysis
* Age Segment Contribution
* Weekday vs Weekend Analysis
* Top Users Analysis
* Business Insights Panel

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling
* Data Visualization

## Key Insights

* Weekday transactions contribute the majority of payment activity.
* Loan services generate significant transaction value.
* User engagement remains consistently high across months.
* Payment success rate exceeds 95%.
